# King County Housing Market Analysis & Price Prediction

## Project Overview

This project focuses on analyzing residential property sales data from King County, Washington, USA, and developing machine learning models to predict housing prices. The analysis covers data cleaning, exploratory data analysis (EDA), feature engineering, and predictive modeling to uncover key factors that influence property values.

The project was completed as part of the IBM Data Science Professional Certificate, where I applied data science methodologies and machine learning techniques to a real-world housing dataset.

---

## Project Goals

* Explore and understand housing market data
* Clean and preprocess raw datasets
* Address missing and inconsistent values
* Perform exploratory data analysis (EDA)
* Identify relationships between housing characteristics and sale prices
* Build and evaluate regression-based machine learning models
* Compare model performance using statistical metrics
* Generate insights that support data-driven decision-making

---

## Dataset Description

The dataset contains residential property sales information from King County, including various structural and geographical attributes such as:

* Property price
* Number of bedrooms and bathrooms
* Living area size
* Waterfront availability
* Property condition and grade
* View quality
* Geographic coordinates
* Lot size and additional housing features
